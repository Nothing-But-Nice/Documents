# Ak'Zam - Item Search & Best in Slot lookup

## Item Search
```
!zam <search string>
```

Example:
```
!zam Dagas
```


## Lookup `Best In Slot (BIS)` items for your class 
```
!bis <race> <class> <slot> [expansion] (Default expansion: The Ruins of Kunark)
```

Example:
```
!bis ogre warrior chest
```
